# Navigations : 1.
Tag (Navigation Section) : The
element defines a section that contains navigation links. 2. Tag (Anchor) : The element is used to create hyperlinks. It can link to external pages, internal page regions, other internal pages, and downloadable files. Types of Navigation in HTML : 1. Target Attribute : The target attribute specifies where to open the linked document. Common values include "_blank" (opens in a new tab) and "_self" (opens in the same tab). 2. Linking to External Pages : Use the tag with the href attribute set to the URL of the external page. 3. Linking to Internal Page Regions : Use the tag with the href attribute set to the ID of the target element on the same page (e.g., href="#section1"). 4. Linking to Internal Pages : Use the tag with the href attribute set to the URL of the internal page (e.g., href="page2.html"). 5. Linking to Downloadable Files : Use the tag with the href attribute set to the URL of the file to be downloaded (e.g., href="file.pdf"). Document
